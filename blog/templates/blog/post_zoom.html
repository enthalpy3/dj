이미지 줌 코딩중...고민거리

  • input 태그를 이용해서 파일을 load할 수 있다.loading 된 파일을 새로운 인스턴스 객체에 담아서 그결과를 저장하고, 저장된 결과물을 id 값에 전달하여 이미지를 화면에 보여줄수 있다.
  • image zoom 함수에서 result.style.backgroundImage = "url('" + img.src + "')"; 부분 img.src 를 가져 오지 못하여 , 요부분에 myimage.onload 를 이용해서 이미를 불러올수있게 해결함.
  • 태그의 속성 중 id 와 class 를 이용해서 이미지를 마우스로 move 할때만 확대 이미지를 볼수 있게 설정함.
  • 파일에 타입을 판단해서 이미지만 불러온다. 다른 파일이면 경고창.
